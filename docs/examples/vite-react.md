## Running the Example Locally

Clone the repo

```sh
git clone https://github.com/fedimint/fedimint-web-sdk.git
cd fedimint-web-sdk
```

Install the package manager with [Corepack](https://nodejs.org/api/corepack.html) & install dependencies.

```sh
corepack enable
pnpm i
```

Run the Example

```sh
pnpm run dev:vite
```
